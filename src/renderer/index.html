Shortcuts Guide

Here is a list of all global keyboard shortcuts available in Imposter:

  • Ctrl + Shift + Enter: Trigger Text Search
  • Ctrl + Shift + B: Toggle Voice Recording Window
  • F10: Fetch Last Voice Transcription & Ask AI
  • Ctrl + Shift + I: Focus input box
  • Ctrl + Shift + S: Screen Snip / OCR
  • Ctrl + Shift + Q: Quit Application
  • Ctrl + Up/Down/Left/Right: Move the window
  • Ctrl + = / -: Resize window
  • Ctrl + Shift + D: Toggle Developer Tools
