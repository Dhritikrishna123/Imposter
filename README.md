## 👻 The "Stealth Mode" Philosophy

**Imposter** isn't just another chat app—it's a ghost on your machine. Built on a unique **Stealth Architecture**, it remains hidden, non-intrusive, and computationally isolated until you trigger the **Systematic Override**.

### Core Stealth Features:
*   **Frameless Ghost Window**: No title bars, no borders, no traditional UI baggage. It feels like part of your wallpaper.
*   **Dynamic Transparency**: A 90% opacity "Glassmorphism" layer that blends seamlessly into any background.
*   **Content Shield**: Integrated protection prevents the app's contents from appearing in screenshots or screen-sharing sessions.
*   **Global Hotkey Orchestration**: Everything is controlled via a series of hidden global shortcuts, keeping your hands on the keyboard and your presence undetected.
*   **Systematic Override**: A custom AI persona engine that prepends your specific rules to every local LLM query, ensuring the responses are perfectly tailored to your "broken system."

---

## 🛠️ How it Works

Imposter operates as a high-performance **Electron-based SaaS shell** talking directly to your local **Ollama** instance. All data—including your name and AI persona—is stored **100% locally** on your device. Nothing ever leaves your machine.

## 🚀 Installation & Setup

### Prerequisites
*   **[Ollama](https://ollama.com/)**: Must be running on port `11434`.
*   **Node.js**: LTS version recommended.

### Local Development
1.  **Clone the Architecture**:
    ```bash
    git clone https://github.com/Puskar-Roy/imposter.git
    cd imposter
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Boot the System**:
    ```bash
    npm start
    ```

### Production Build (.exe)
To generate a standalone Windows executable for your own use:
```bash
npm run dist
```
Find your generated files in the `/dist` folder.
